Since i am programmer i will start by identifying what is the problem so i can solve it, Sorting is the problem if representing numbers or alphabetic in the wrong order and your goal is to make them in the right order and i think it's the best problem to start solving using a simple alogorithm.

First algorithm: Bubble sort

it's a simple sorting algorithm that repeatedly steps through the list to be sorted, compares each pair of adjacent items and swaps them if they are in the wrong order. The pass through the list is repeated until no swaps are needed, which indicates that the list is sorted

public class App {
    public static void main(String[] args) {

        /// variables ///
        int[] sort_me = {6, 5, 3, 1, 8, 7, 2, 4};
        long startingTime, elapsedTime;

        /// let's print the array before sorting ///
        System.out.println("ARRAY BEFORE SORTING");
        for (int i = 0; i < sort_me.length; i++) {
            System.out.println("Array[" + i + "]= " + sort_me[i]);
        }

        System.out.println(); // let's make it look good

        startingTime = System.nanoTime();

        // Bubble sort algorithm ///

        for (int operation_number = 0; operation_number < sort_me.length - 1; operation_number++) {
            for (int element = 0; element< sort_me.length - 1; element++ ) {
                if (sort_me[element] > sort_me[element + 1]) {

                    /// Swapping ///
                    int temporary = sort_me[element];
                    sort_me[element] = sort_me[element + 1];
                    sort_me[element + 1] = temporary;
                }
            }
        }

        elapsedTime = System.nanoTime() - startingTime;


        /// let's print the array After sorting ///
        System.out.println("ARRAY AFTER SORTING");
        for (int i = 0; i < sort_me.length; i++) {
            System.out.println("Array[" + i + "]= " + sort_me[i]);
        }

        System.out.println("Time required : " +elapsedTime/6 + " Millisecond");





    }
}
